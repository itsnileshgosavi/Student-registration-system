Student Registration

This student registration site is a web-based application developed using HTML, Tailwind CSS, and JavaScript. This site utilizes a local storage to store student data, and shows the registration records. You can enter the details of the students in the form provided and click on submit button. The information entered is then saved into your local storage. You can Check the stored data anytime. The system also validates the information provided and if it matches the requirements then only allows the submission.
